Some More Fitness Tests

Here are some more tests that may be added to this site. These are tests I have recently come across and for some limited information is available. Let me know if there are others that you cannot find information about.

  • AAHPERD basketball skill test
  • Agility 9-3-6-3-6-9
  • Anaerobic Cycling Test (MACT) - a modification of the MART.
  • Canadian fitness test
  • Danish step test 
  • Davies Closed Kinetic Chain Upper Extremity Stability Test
  • Death by 10m (a crossfit workout, not really a fitness test)
  • Eye-leg co-ordination test.
  • Four Minute O'Neill Fitness Test - a general aerobic fitness testing using the Concept2 Indoor Rower.
  • Functional Threshold Power or Pace (FTP) test - a simple way of determining your current cycling performance level. It is the highest average power you can sustain for an hour, measured in watts. FTP is often used to determine training zones when using a power meter.
  • Hoff Tests
runners
  • Incline Pull Up Test - alternative to the NAPFA Pull-Up Test. Compare to the horizontal pull up test.
  • Inverted balance test.
  • Isometric Mid-Thigh Pull (IMTP)
  • Latham Reaction Time test
  • Mantoux test for tuberculosis diagnosis.
  • MRI age test
  • Nelson's hand reaction time test (Nelson's Speed of Movement Test)
  • Nelson's leg reaction time.
  • OmegaWave Work Capcity test
  • Ohio Step Test
  • ORPAT the oregon physical abilities test - a hybrid physical ability/job sample physical abilities assessment process designed to evaluate police officer candidates on the essential physical capacities required to satisfactorily perform their job duties.
  • Progressive Pulse Ratio Test (old YMCA test) - step test
  • Probst Field Test - intermittent test for soccer players including change of directions.
  • S180 degrees agility test
  • Speed Court - a 7m by 7m floor with 10-12 contact plates. This test can be used to measure reaction times, footwork, agility, cognition, acceleration, and direction changes.
  • Speed tests - test of the distances covered in 4.6 or 8 seconds (no other reference can be found)
  • Squat thrust test.
  • Single-Arm Seated Shot Put
  • Tuttle Step Test
  • Ear-hand or ear-foot coordination tests, for blind players (response to sound stimulus)
