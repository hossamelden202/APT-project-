GeneXproTools Modeling Frameworks
  Logistic Regression Software
A hybrid algorithm that brings together the power of standard logistic regression and evolutionary modeling. The core models are created using nonlinear evolutionary techniques and, therefore, are usually much more accurate than the simple logistic regression model.

Also based on nonlinear methods and functions, the classification algorithms of GeneXproTools can automatically find accurate decision boundaries to recognize the patterns in your data. And by combining different models, you can easily create model ensembles with as many models as you wish.

  Time Series Prediction Software
Allows not only the design of nonlinear forecasting models from univariate time series data but also recursive prediction. Time series data can also be modeled using the nonlinear regression techniques of the Regression Framework of GeneXproTools.

  Logic Synthesis Software
A different approach to the automatic design and optimization of logic circuits, using all kinds of logical functions (a total of 258 built-in logical functions), including the canonical Boolean functions AND, OR, and NOT. Itís also an essential tool for simplifying all kinds of logic circuits no matter the gates used to build them.
