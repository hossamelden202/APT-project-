Enum log::LogLevel [?] [src]

#[repr(usize)]

pub enum LogLevel {
    Error,
    Warn,
    Info,
    Debug,
    Trace,
}

An enum representing the available verbosity levels of the logging framework.

Typical usage includes: checking if a certain LogLevel is enabled with log_enabled!, specifying the LogLevel of log!, and comparing a LogLevel directly to a LogLevelFilter.

Variants

Error

The "error" level.

Designates very serious errors.

Warn

The "warn" level.

Designates hazardous situations.

Info

The "info" level.

Designates useful information.

Debug

The "debug" level.

Designates lower priority information.

Trace

The "trace" level.

Designates very low priority, often extremely verbose, information.
