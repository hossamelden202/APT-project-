System Requirements

  • Operating System: macOS 10.6 Snow Leopard or later, including macOS 15 Sequoia
  • Display: 1024 x 768, millions of colors
  • Free Hard Disk Space: 4.83 GB
  • DVD or USB Drive: For installation only if ordering a copy on DVD or USB.
Import Formats:
PDF, JPG, TIFF, PICT, PNG, BMP, WMF, GIF, PSD, TGA, MAC, SGI.
Export Formats for Projects:
PDF, JPG, TIFF, PICT, PNG, BMP, HTML.
Language Support:
Most major European, Cyrillic and Asian languages can be used to create Headlines and text blocks. Hebrew and Arabic are not fully supported.
Label Stock Support:
US Letter and A4 (Avery, CD Stomper, Memorex and NEATO brands)
Paper Size Support:
All U.S. sizes and International sizes (A4, A3, JB5).
Compatibility:
Fully compatible with Software MacKiev editions of The Print Shop (The Print Shop for Mac, The Print Shop for Mac 2 and The Print Shop for Mac 3). Mostly compatible with The Print Shop Deluxe 1.1.3 for Mac and The Print Shop for Windows versions 6 through 23.
