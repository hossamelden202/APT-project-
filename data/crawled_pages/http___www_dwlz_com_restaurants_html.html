Restaurants Points

*Whenever possible, I calculate ALL my Points on my website with
my Offline Points Calculator in the Half Points Mode,
which rounds UP to the nearest Half Point.
This keeps my site consistent. 
All point values in black boldare the new PointsPlus values,
while all point values in red boldare the old Point values.



I am aware that there are no half Points on the new plan, or old one, for that matter. I wish to be consistent with my
values throughout the site. I find myself adding additional Points to my meal totals when I eat out, so being up a
half Point is most likely a good thing, if you know what I mean. This is how I've always figured Point values
and will continue to do so.  Thank you for understanding.
