About the Sun In Time

For any date on which SDO/AIA has been observing the Sun, you may view a set of the observable wavelength channels, an SDO/HMI magnetogram, and blends of these channels. The iSolSearch tool displays a summary of reported events that occurred on this date, followed by light curves (counts in each of the wavelength channels as a function of time). The light curve plots show Sun-integrated trends in the various channels. The imagery shown on this webpage is generally in the public domain, however please refer to the "Rights/Usage" statement at bottom of the webpage for the requested acknowledgement. Further questions can be directed to hek_support@lmsal.com.
