Other Java SE products

Oracle Java SE Advanced, Java SE Advanced Desktop, Java SE Suite, Java SE Subscription, and Java SE Desktop Subscription products have transitioned to Java SE Universal Subscription to provide more flexibility. Customers who have Java SE Advanced, Java SE Advanced Desktop, Java SE Suite, Java SE Subscription, Java SE Desktop Subscription, and/or use Oracle Java SE with an Oracle product will continue to receive support and updates as usual. New or existing customers who want commercial Java SE licenses and support can now choose Java SE Universal Subscription.
