State Information:

State Motto: "Eureka" (I have found it)

State Animal: California grizzly bear (Ursus californicus)

State Bird: California Quail (Lophortyx californica)

State Colors: Blue and Gold

State Dance: West Coast Swing Dancing

State Fife and Drum Band: California Consolidated Drum Band

State Fish: The golden trout (Salmo agua-bonita)

State Flower: California Golden Poppy (Eschsholtzia californica)

State Folk Dance: Square Dance

State Fossil: The sabre-tooth cat (Smilodon californicus)

State Gemstone: Benitoite

State Historical Society: California Historical Society

State Insect: The California dogface butterfly (Zerene eurydice)

State Marine Fish: the garibaldi (Hypsypops rubicundus)

State Marine Mammal: the California gray whale (Eschrichtius robustus)

State Mineral: Gold

State Prehistoric Artifact: the chipped stone bear

State Reptile: the desert tortoise (gopherus agassizi)

State Rock: Serpentine

State Soil: The San Joaquin Soil

State Song: I Love You, California, Lyrics by F.B. Silverwood and music by A.F. Frankenstein, published 1913

State Tartan: pattern and colors of the John Muir family

State Theater: Pasadena Playhouse

State Tree: California Redwood - the coast redwood (Sequoia sempervirens) and the giant sequoia (Sequoia gigantea)

State Capital: USA City Directory>Sacramento

State Flag

Size: The third largest state, 163,707 square miles

Statehood: the 31st State. Admitted Sept. 9, 1850
