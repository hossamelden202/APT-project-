State Information:

Motto: Gold and Silver

State Animal: Grizzly Bear, adopted 1983

State Ballad: Montana Melody by Carleen Harvey and LeGrande Harvey, adopted 1983

State Bird: Western Meadowlark, adopted 1931

State Butterfly: Mourning Cloak, adopted 2001

State Fish: Blackspotted Cutthroat Trout, adopted 1977

State Flower: Bitterroot, adopted 1895

State Fossil: Duck-billed Dinosaur, adopted 1985

State Gem: Sapphire, Montana Agate, adopted 1969

State Grass: Bluebunch Wheatgrass, adopted 1973

State Language: English, adopted 1995

State Song: Montana, lyrics by Charles Cohan, Music by Joseph E. Howard, adopted 1945

State Tree: Ponderosa Pine, adopted 1949

State Capital: USA City Directory>Helena

State Flag

Size: The 4th largest state, 147,046 total square miles

Statehood: the 41st State. Admitted Nov. 8, 1889
