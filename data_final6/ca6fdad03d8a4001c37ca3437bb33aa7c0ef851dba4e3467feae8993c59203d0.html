2010 Medal Tally

Medal count for the Winter Games of 2010

We have listed this table twice. Firstly using the standard ranking system based on the number of gold medals won, then silver and bronze. Another table is also shown, listing the order based on the total number of medals won. The main reason to show both tables is that they produce a different medal tally leader.

At the Vancouver Games, the USA had the most number of medals in total, while both Canada and Germany are above them when ranked by the most number of gold medals. So depending on which way you think the order should be ranked, there is a different winner. There is no official Olympic Games method - see more about Medal Ranking Systems.
